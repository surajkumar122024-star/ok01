# **App Name**: OpticShift Pro

## Core Features:

- Local Image Compression: Reduces file size of images using browser-side processing, ensuring no data ever leaves the user's device.
- Dynamic Image Resizing: A responsive interface for altering dimensions while maintaining aspect ratios through Canvas-based rendering.
- Lossless Format Conversion: Bi-directional conversion between JPG and PNG formats performed entirely within the client environment.
- Zero-Latency Preview Engine: Immediate visual feedback using Object URLs for transformed images prior to download.
- Ad-Ready Component Library: Layout includes dedicated reserved areas optimized for Google AdSense placement and high viewability.
- SEO & Meta Infrastructure: Built-in automated sitemap generation, robots.txt, and per-page Open Graph metadata for organic growth.

## Style Guidelines:

- Primary color: High-trust Cobalt Blue (#1E70FF) to convey professional reliability.
- Background color: Ultra-clean Frost (#F0F4F8), providing a subtle desaturated tint from the primary hue.
- Accent color: Ocean Cyan (#14B0CF) for call-to-action buttons and conversion status indicators.
- Body and Headline font: 'Inter' sans-serif for its machined, objective look and exceptional readability on mobile devices.
- Clean, thin-line monochromatic icons that complement the professional SaaS aesthetic without adding visual weight.
- Glassmorphic dashboard containers with 12px blur and soft borders for a modern, layered appearance.
- Gentle state transitions (150ms ease-in-out) for tool interactions and file drag-and-drop feedback.
